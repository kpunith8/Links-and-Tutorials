### Install fonts on Mac

- Install the Fira Code

```
$ brew tap caskroom/fonts
$ brew cask install font-fira-code
```

## Mac Shortcuts

- `cmd+Ctrl+Q` - Lock
- `cmd+Shift+F` - Toggle full screen
- `cmd+C` - Copy
- `cmd+Option+V` - Cut/Paste
